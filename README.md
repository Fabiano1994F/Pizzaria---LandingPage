# 🍕 SP Pizzaria - Sistema de Reservas e Cardápio Online

Este é um projeto moderno de landing page para uma pizzaria, desenvolvido com foco em **experiência do usuário (UX)**, **responsividade** e **performance**. O site conta com um slider dinâmico, cardápio interativo e um sistema de reservas direto.

## 🚀 Tecnologias Utilizadas

Este projeto foi construído utilizando as seguintes tecnologias:

* **HTML5** - Estrutura semântica do projeto.
* **CSS3** - Estilização moderna com Flexbox, Grid e animações.
* **JavaScript (Vanilla)** - Lógica para o menu mobile e funcionalidade do slider.
* **Google Fonts** - Tipografia personalizada (Poppins).

## 📱 Funcionalidades

- [x] **Design Responsivo:** Adaptado para dispositivos móveis, tablets e desktop.
- [x] **Slider Dinâmico:** Destaques da pizzaria com transições suaves.
- [x] **Cardápio Interativo:** Grade de produtos com efeitos de hover e botões estilizados.
- [x] **Sistema de Reserva:** Formulário de contato pronto para integração com backend.
- [x] **Menu Hamburguer:** Navegação otimizada para usuários mobile.
